#面向对象的特性

JavaScript面向对象的特性

##封装性
对象是将数据与功能组合到一起, 即封装

1. js 对象就是 键值对的集合
 * 键值如果是数据( 基本数据, 复合数据, 空数据 ), 就称为属性
 * 如果键值是函数, 那么就称为方法
2. 对象就是将属性与方法封装起来
3. 方法是将过程封装起来

##继承性

所谓继承就是自己没有, 别人有,拿过来为自己所用, 并成为自己的东西

1. 传统继承基于模板
2. js 继承基于对象
js继承实现举例：混入（mix）

```js
function mix ( o1, o2 ) {
    for ( var k in o2 ) {
        o1[ k ] = o2[ k ];
    }
}
```
